## 1. 認証・認可

| 項目 | 方針 |
|------|------|
| 認証トークン | JWT を HTTPOnly Cookie に保存（XSS対策） |
| CSRF対策 | SameSite=Strict Cookie |
| 認可 | 全APIエンドポイントでJWT検証 + リソースの所有者確認 |

## 2. パスワード管理

| 項目 | 方針 |
|------|------|
| ハッシュアルゴリズム | bcrypt |
| コスト係数 | 12以上 |

## 3. 入力バリデーション

| 項目 | 方針 |
|------|------|
| バリデーションライブラリ | Zod |
| 実施タイミング | サーバーサイド（APIエンドポイント受信時） |
