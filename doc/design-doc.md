# TODOアプリ 設計書

## 1. 概要

### 1.1 目的
ブラウザ上で動作するTODO管理アプリ。複数のユーザーがそれぞれ自分のタスクを管理できる。

### 1.2 スコープ
- ユーザー登録・認証
- タスクのCRUD操作（作成・参照・更新・削除）
- タスクへの属性付与（期限・優先度・タグ・担当者）
- リスト表示とカンバンボード表示
- ダッシュボードでの期限追跡

### 1.3 スコープ外
- プロジェクト・グループ管理
- リアルタイム同期
- プッシュ通知・メール通知
- 他ユーザーへのタスク共有・委譲

---

## 2. 機能要件

### 2.1 認証
| # | 機能 | 説明 |
|---|------|------|
| A-1 | ユーザー登録 | メールアドレス・パスワードで新規登録 |
| A-2 | ログイン | メールアドレス・パスワードで認証 |
| A-3 | ログアウト | セッション・トークンを無効化 |
| A-4 | パスワードリセット | メールによるパスワードリセット（将来対応） |

### 2.2 タスク管理
| # | 機能 | 説明 |
|---|------|------|
| T-1 | タスク作成 | タイトル・説明・期限・優先度・タグを設定して作成 |
| T-2 | タスク一覧 | 自分のタスクを一覧表示（リスト・カンバン切替） |
| T-3 | タスク編集 | 属性の変更 |
| T-4 | タスク削除 | タスクの削除（論理削除） |
| T-5 | ステータス変更 | Todo / In Progress / Done の変更（カンバンのドラッグ&ドロップ含む） |
| T-6 | タスク検索・フィルタ | タグ・優先度・期限・ステータスでフィルタリング |

### 2.3 ダッシュボード
| # | 機能 | 説明 |
|---|------|------|
| D-1 | 期限切れタスク表示 | 期限が過ぎたタスクを強調表示 |
| D-2 | 今日期限のタスク表示 | 当日期限のタスクを一覧表示 |
| D-3 | タスクサマリ | ステータス別・優先度別のタスク数を表示 |

---

## 3. 非機能要件

| 項目 | 要件 |
|------|------|
| 対応ブラウザ | Chrome, Firefox, Safari, Edge（最新版） |
| レスポンシブ | デスクトップ優先（モバイル対応は将来） |
| 認証セキュリティ | JWT + HTTPOnly Cookie |
| パスワード | bcrypt でハッシュ化 |
| API通信 | HTTPS |
| データ保護 | ユーザーは自分のデータのみアクセス可能 |
